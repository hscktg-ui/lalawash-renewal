# 패널 30인 — 브랜드 노출 재점검

> 안건: 로고 깨짐 · 과다 노출 · 파비콘/OG/슬로건 활용

---

## 원인 (기술)

| 파일 | 문제 |
|------|------|
| `logo.png` | **흰 글씨 + 투명 배경** → 흰 헤더에서 안 보임(깨진 것처럼 보임) |
| 히어로 `brightness-0 invert` | 흰 로고를 강제로 반전 → 품질·대비 깨짐 |
| 헤더+히어로+푸터 중복 | 로고 도배로 브랜드가 산만해짐 |

## 패널 합의 (배치)

| 위치 | 결정 | 담당 |
|------|------|------|
| **헤더** | `logo-light.png`(파란 워드마크) **1회만** | P21 BI |
| **히어로** | 로고 이미지 **제거** → 슬로건「지속가능한 깨끗함」+ 브랜드명 타이포 | P23 AD · P15 |
| **푸터** | 이미지 로고 **제거** → 텍스트「라라워시」+ 슬로건 | P22 UI |
| **파비콘** | 기존 `favicon.ico` / `favicon-196` 유지 | P21 |
| **미리보기(OG)** | 기존 `og-image.png`(슬로건 포함) 유지 · 메타 타이틀에 슬로건 | P16 SEO |
| **파트너 영역** | 라라워시 로고 추가 금지(파트너만) | P30 CD |

## 슬로건 활용

- 공식: **지속가능한 깨끗함** (OG·기존 사이트와 동일)
- 타이틀: `라라워시 | 지속가능한 깨끗함`
- 히어로 eyebrow / 푸터 / OG description에 일관 사용

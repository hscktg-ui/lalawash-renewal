## 1. 브랜드 한 줄 정의

**라라워시**는 경기도 기반 다회용기 **렌탈·수거·세척·재공급** 순환 인프라 브랜드입니다.  
식판·컵·축제·장례까지 현장형 솔루션을 제공하며, 자활·사회적경제와 결합해 **위생 + 환경 + 일자리**를 동시에 만듭니다.

## 3. 벤치마크 인사이트

### 더그릿 (광고주 강조 · Primary)
- 미션 히어로 + **실시간 임팩트 숫자** (탄소·용기·나무)
- 제품/디바이스/세척/플랫폼 **솔루션 모듈화**
- 인프라 규모(세척량·센터 수)를 전면에
- 인증·수상으로 Trust 마감

**라라워시 적용:** 감성 카피보다 “숫자로 증명하는 순환 인프라” 톤. 단, 라라워시만의 **지역 밀착·자활 일자리·식판 전문성**은 차별 포인트로 유지.

### 클린앤키즈
- 위생 테스트(ATP 등)와 장비 스택을 명시
- “프랜차이즈 식판세척” 포지셔닝 명확

**적용:** 6단계 세척 + ATP + UPW를 홈 중앙에.

### 뽀득 / ESG커넥트 / 세이브어스 / 리턴미
- B2B 전환 경로 단순, ESG·임팩트 스토리텔링
- 서비스 플로우를 다이어그램으로 설명

**적용:** 7단계 Clean Circle, ESG 성과 섹션.

## 3. 현재 사이트 맵 (합의된 배치)

```
/                 홈 (임팩트·인프라·솔루션·위생·성과·블로그·문의)
/about            소개
/services         서비스 허브
/services/:slug   급식·공공·축제·장례
/infra            지점 (기존 center 지도·주소·전화)
/impact           성과·연도별 임팩트
/notice           공지·공문 (사이트)
/notice/:id       공지 상세
/contact          문의
/admin            관리자 로그인 (메뉴 비노출)
/admin/notices    공지 등록·수정·삭제
```

**채널 분리 (패널 합의)**  
- 일상 소식 → **네이버 블로그**  
- 공식 공지·공문 → **사이트 `/notice`**

## 7. 잘된 점 (패널이 남긴 강점)

1. **한 화면 설득**: 숫자 → 인프라 → 솔루션 → 위생 순서가 공공 담당자에게 읽힘  
2. **사실 보수 운영**: 미검증 행사 사진 미귀속, 지점 정보는 원사이트 사용  
3. **브랜드 절제**: 로고 도배 금지, 슬로건·OG가 브랜드를 보조  
4. **운영 채널 분리**: 소식=블로그, 공문=사이트 공지  
5. **라라워시 차별**: 전국 테크 카피가 아닌 **가까운 세척망·자활**

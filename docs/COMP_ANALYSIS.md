## 2. 경쟁사 철저 비교

### 2-1. 더그릿 (Primary · 광고주 강조)

| 패턴 | 더그릿 | 라라워시 v1 | 갭 |
|------|--------|-------------|-----|
| 히어로 | 미션 한 방 (“We move the world…”) | 감성 카피 + 통계 3개 | 미션 임팩트·타이포 스케일 부족 |
| Impact | 실시간 대형 카운터 3종 (탄소·용기·나무) | 섹션은 있으나 밀도·연출 약함 | **핵심 갭** |
| 인프라 | **일 65만개**, 직영3+가맹6, 면적 | 18지역·20사업장 나열 | **용량(capacity) 헤드라인 없음** |
| 솔루션 | drimm container/device/wash/platform 제품화 | 4카드 나열 | **제품/시스템 네이밍·모듈감 부족** |
| Trust | 녹색기술·ISO·에코라벨·베이비유니콘 | 경기도·자활 로고만 | **인증·수상 밴드 부재** |

| CTA | 앱·가이드·Learn more 계층 | 상담 중심 | OK (B2B면 상담이 맞음) |

### 2-2. 클린앤키즈

- 카피 훅: “작은 식판에도 큰 위생”
- **위생 테스트 4종** 명시 (ATP, pH, 고온라벨, 잔류세제)
- **장비 스택** 명시 (컨베이어·초음파·브러쉬·열풍)
- 증명: 거래처 800, KBS, 폐업 ZERO  
→ 라라워시: ATP·UPW는 있으나 **테스트/장비 갤러리·증명 밀도** 부족

### 2-3. 리턴미

- 슬로건: “쓰고 버리는 시대 → 쓰고 돌려주는 시대”
- **연도별 임팩트**(’23/’24) + **산출 공식 공개**
- 파트너 로고 월 대량
- 제품 리스트·요금/플랜 진입점  
→ 라라워시: 누적 숫자만, **연도 비교·산출식·파트너월** 없음

### 2-4. ESG커넥트(이벤저스/포아브)

- 타깃 CTA 분리: 축제/상조/식판
- 행사 포트폴리오(실사·건수)가 스크롤의 주인공  
→ 라라워시: 축제 50건 실적은 있으나 **갤러리형 포트폴리오** 없음

### 2-5. 뽀득 / 세이브어스 (타임아웃·보조)

- B2B·구독·라이프스타일 전환 UX가 단순하고 CTA가 분명  
→ 라라워시 문의 폼은 있으나 **견적 시나리오(몇 컵/며칠)** 없음

### 2-6. 기존 라라워시 (문제 재확인)

1. GNB 과다(소개·캐릭터·협력사·시스템…)  
2. 히어로가 배너/예약 유도로 산만  
3. 임팩트 숫자 부재(회사소개서엔 있음)  
4. 서비스가 페이지 파편화  
5. 지점 텍스트 나열  
6. B2B 신뢰(인증·용량·테스트) 약함  
7. 푸터·지점 중복  
8. 캐릭터 비중이 B2B 톤과 충돌

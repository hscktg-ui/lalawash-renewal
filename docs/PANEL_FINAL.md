## 2. 실사 포트폴리오 노출 정책 (합의)

### 문제
CDN에 있는 일반 운영 사진을 **특정 축제·기관 실사**처럼 붙이면 사실 오류·신뢰 리스크가 큼.

### 패널 결정

| 구분 | 노출 | 근거 |
|------|------|------|
| 회사소개서 수치·기관명 | **노출** | v12에 명시된 검증 정보 |
| 특정 행사명 + 임의 사진 귀속 | **비노출** | 원본 사진 미확보 |
| 현장 포토 갤러리(귀속형) | **보류** | `showAttributedPhotos: false` |
| 파트너 공식 로고(경기도·자활) | **노출** | 기존 사이트 사용 중인 에셋 |
| 파트너 워드마크(시청명 등) | **기관명 텍스트** | 로고 파일·승인 미확인 |

**코드:** `PORTFOLIO_POLICY` + `VERIFIED_ACHIEVEMENTS`  
**검수:** P03(축제) · P02(공공) · P04(장례) · P12(ESG) · P15(카피) · P30(CD) → **Go (조건부: 원본 사진 후 갤러리 오픈)**

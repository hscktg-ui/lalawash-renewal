| 항목 | 데이락 | 기존 라라워시 | 리뉴얼(현재) | 패널 결론 |
|------|--------|---------------|--------------|-----------|
| 히어로 | 풀블리드·혜택 카피 롤 | 슬로건+예약 CTA | 브랜드명+실무 설명 | **혜택 한 줄 강화**, 첫 화면 통계는 축소 |

| 상거래 | BEST/CART | 없음 | 견적폼·몰 링크 | **견적 CTA 유지**, 카트 UX 비채택 |
| 호흡 | 여백·비주얼 | 블록 나열 | 정보 밀도 높음 | **스토리 뒤 숫자**로 순서 조절 |

### 데이락에서 가져올 것
1. **한 문장 혜택**이 히어로를 이끈다  
2. **브랜드 스토리**가 감정을 연다 (일상·편안함 톤을 공공 언어로 번역)  
3. **3개 가치**로 기억시킨다  
4. 풀블리드·여백으로 **호흡**을 준다  

### 데이락에서 가져오지 않을 것
- 추천상품·장바구니·가격 노출 그리드  
- D2C 리뷰 별점 UI (대신 기관 신뢰 유지)  

### 기존 라라워시에서 지킬 것
- 예약/상담 CTA, 지점 밀착, 현장 실사 톤

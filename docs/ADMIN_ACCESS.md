# 라라워시 관리자 접속 정보

> 외부에 공유하지 마세요. 운영 담당자만 보관하세요.

## 접속 주소

- 관리자 로그인: `/admin`
- 공지 관리: 로그인 후 자동 이동
- 공개 공지 게시판: `/notice`

## 계정

| 항목 | 값 |
|------|------|
| 아이디 | `lalawash` |
| 비밀번호 | `LalaWash#Notice2026` |

## 사용 방법

1. `https://(배포주소)/admin` 접속
2. 아이디·비밀번호 입력
3. 공지 등록 / 수정 / 삭제
4. 「공문」 유형을 고르면 게시판에 공문으로 표시됩니다

## 참고

- 공지 내용은 현재 브라우저(localStorage)에 저장됩니다.
- 다른 PC에서도 동일하게 쓰려면 추후 DB 연동이 필요합니다.
- 비밀번호 변경이 필요하면 개발팀에 요청하세요.

# 라라워시 관리자 접속 정보

> 외부에 공유하지 마세요. 운영 담당자만 보관하세요.

## 접속 주소

- 관리자 로그인: `/admin`
- 공지 관리: 로그인 후 자동 이동
- 공개 공지 게시판: `/notice`

## 계정

| 항목 | 값 |
|------|------|
| 아이디 | `lalawash` |
| 비밀번호 | `LalaWash#Notice2026` |

## 사용 방법

1. `https://(배포주소)/admin` 접속
2. 아이디·비밀번호 입력
3. 공지 등록 / 수정 / 삭제
4. **이미지 첨부**: 사진 선택(여러 장) 또는 이미지 URL · 대표 이미지 지정 · 발행 미리보기
5. 「공문」 유형을 고르면 게시판에 공문으로 표시됩니다

## 참고

- 공지 내용은 현재 브라우저(localStorage)에 저장됩니다. (`lalawash_notices_v5`)
- 이전 홈페이지 공지 **22건(이미지 포함)** 은 시드로 이관되어 있습니다. 새 브라우저·v5 전환 시 자동 반영됩니다.
- 관리자 **「시드·이미지 동기화」** 로 레거시 글을 최신 시드로 다시 맞출 수 있습니다. (직접 작성한 `n-*` 글은 유지)
- 이미지는 자동 압축 후 저장됩니다. (최대 6장/글) 용량이 부족하면 장수를 줄여 주세요.
- 다른 PC에서도 동일하게 쓰려면 추후 DB 연동이 필요합니다.
- 비밀번호 변경이 필요하면 개발팀에 요청하세요.
